# HeyGen (Avatar Mode)

Optional “WOW” feature: real-time avatar for instructor interventions.

## Responsibilities

- **Re-explain mode** — When instructor triggers “Generate 30-sec re-explain,” HeyGen Avatar can read the generated script with low latency
- **Instructor-triggered intervention** — Avatar delivery is optional and controlled by the instructor

## Tech stack

- HeyGen Avatar API

## Environment

- `HEYGEN_API_KEY`
