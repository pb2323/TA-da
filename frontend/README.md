# Frontend (Zoom × Vercel)

Zoom in-meeting app and Next.js frontend for TA-DA.

## Responsibilities

- **Zoom App** — In-meeting side panel: concept cards, confusion signals (Lost / Kinda / Got it), chat, and instructor controls
- **Next.js** — Main web app; deployed on Vercel
- **Real-time UI** — Streamed responses and live updates from the backend

## Tech stack

- Zoom SDK / App framework
- Next.js
- Vercel (hosting)

## Related

- Backend WebSocket and API: [backend](../backend/)
- Zoom credentials: root `.env.local` (`ZOOM_CLIENT_ID`, `ZOOM_CLIENT_SECRET`)
