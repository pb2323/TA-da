# Fetch.ai (Multi-Agent Layer)

Multi-agent workflow and Agentverse integration for TA-DA.

## Responsibilities

- **Moments Agent** — Lecture moments, segmentation, context
- **Confusion Agent** — Confusion detection, clustering, intervention suggestions
- **Tutor Agent** — Multi-turn diagnostic tutoring (clarify → explain → check → adapt)
- **Contract Agent** — Learning contract generation and personalization

Agents are registered on **Agentverse** and are Chat Protocol compliant.

## Tech stack

- Fetch.ai Agentverse
- Multi-agent orchestration and workflows
